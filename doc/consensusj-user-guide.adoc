=== Generic

The module `consensusj-jsonrpc` contains `JsonRpcClientHttpUrlConnection` which is a generic JSON-RPC client.

The module `cj-btc-jsonrpc` contains `BitcoinClient` and `ExtendedBitcoinClient`.
