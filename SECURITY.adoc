= Security Policy

== Supported Versions

As **ConsensusJ** is only available as a _pre-release_, there are no officially supported version(s) of **ConsensusJ** at this time.
However, we do take security seriously and will accept and respond to vulnerability reports.

[cols="1,1"]
|===
| Version | Supported

| < 1.0   | ❌ (see above)
|===

== Reporting a Vulnerability

To report a vulnerability please use **Report a vulnerability** button on the https://github.com/ConsensusJ/consensusj/security[Security] page.
